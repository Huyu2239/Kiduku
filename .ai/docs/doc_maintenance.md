# docs 運用メモ（AI向け）

## 目的

- `docs/` は人間向けの正式ドキュメントを置く
- `.ai/docs/` は AI 向けの補助メモや作業用ノートを置く

## 更新時のチェック

- `docs/index.md` と各サブディレクトリの `index.md` を更新する
- 1ファイルにつき1主題とする
- 未確定事項は `docs/` ではなく `.ai/docs/` に記載する
- 認証情報や秘密情報は、コードおよびドキュメントに含めない
- AI の実装や提案は `docs/` に記載された設計・規約に準拠する

## 注意

- `.envrc` にはクレデンシャルが含まれるため、参照しない
